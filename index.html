🌟 Oerion Language - Developer Docs

This language is open-source and created with ❤️ by Raja Sharma. Feel free to use, modify, and contribute.

📚 Example Code Snippets

let nums = [1, 2, 3] nums.append(4) nums.remove(2) print nums

Creates a list, appends 4, removes 2, and prints the final list.

let name = "Raja" print "Hello", name

Declares a string and prints a greeting with it.

print type nums

Prints the data type of the variable nums.

for i in range(5): if i == 3: break print i

Loops from 0 to 4, prints numbers until 3, then breaks.

try: let x = 5 / 0 except: print "Caught division error!"

Demonstrates basic error handling for division by zero.

let x = "123" to_int(x)

Converts a string to an integer.

import "utils.oer"

Imports utility functions from an external Oerion script file.

📤 Output

print "Hello"

Displays text in the console.

print myVar

Prints the value of myVar.

🧠 Variables

let x = 5

Creates a variable x with value 5.

let name = "Raja"

Stores the string "Raja" in name.

🔎 Conditions

if x > 3:
    print "x is greater than 3"

Checks a condition. Runs the block if true.

🔁 Loops

while x < 5:
    print x
    let x = x + 1

Repeats the block while condition is true.

for i in range(3):
    print i

Repeats block 3 times, with i from 0 to 2.

🧾 Input

let name = input("Enter your name: ")

Prompts the user and stores input in name.

🛠️ Built-in Tools

type myVar

Returns the type of myVar (like int, str).

to_int("42")
to_float("3.14")

Converts strings into numbers.

📦 Examples

let a = 10
let b = 20
let sum = a + b
print sum

Adds two numbers and prints the result.

let i = 0
while i < 3:
    print i
    let i = i + 1

Prints 0 to 2 using a while loop.

for x in range(5):
    print x

Prints numbers 0 through 4 using a for loop.
